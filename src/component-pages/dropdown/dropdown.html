드롭다운 메뉴

메모

📣 드롭다운 메뉴 width 값은 2가지 경우로 구분되어 사용됩니다.

  • 1. 아래 컴포넌트와 같이 사이즈(normal, small, xs)별 패딩 값을 적용하는 경우
  • 2. input의 width 값과 동일한 width 값을 적용하는 경우

📣 input을 한 번 클릭한 후의 드롭다운 메뉴 위치가 정확한 위치입니다.

  • 현재 페이지에서 드롭다운 메뉴를 모두 강제로 펼쳐두어서인지, 처음 드롭다운 메뉴의 위치가 살짝 어긋나 보이는 문제가 있습니다.🥲
  • 인풋을 한 번 클릭한 후의 드롭다운 메뉴 위치가 정확한 위치입니다. (현재 페이지에서만 발생하는 문제이며 다른 페이지에서는 발생하지 않음)
